# HelpFromHumans

A web-based platform where users can request help from others.

## Features
- Submit help requests
- View requests
- Simple user interface

## Tech Stack
- Node.js
- JavaScript
- HTML/CSS

## How to Run
1. Install Node.js
2. Run: npm install
3. Run: node index.js
4. Open in browser

## Future Improvements
- Add database
